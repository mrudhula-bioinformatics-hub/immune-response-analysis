# Immune Response Analysis using Python

## Explanation

This project analyzes gene expression data under control and infected conditions using Python and pandas.

Fold change is calculated to understand how gene expression changes during infection.

---

## Results

### Fold Change Summary

| Gene   | Fold Change |
|--------|------------|
| CXCL10 | 13.33 |
| IRF7   | 7.00 |
| IL6    | 5.00 |
| IFNG   | 5.00 |
| TNF    | 4.50 |
| STAT1  | 4.40 |
| CCL2   | 2.00 |
| IL10   | 0.80 |

### Key Observations

- CXCL10 shows the highest increase in expression  
- Several genes show strong upregulation (IL6, IFNG, TNF)  
- IL10 shows reduced expression  

---

## Interpretation

The results indicate a strong immune response during infection.

- CXCL10 is highly upregulated, suggesting immune cell recruitment  
- IRF7 and IFNG indicate antiviral activity  
- IL6 and TNF suggest inflammation  
- IL10 is downregulated, indicating reduced anti-inflammatory regulation  

Overall, the infected condition shows increased gene expression, indicating activation of the immune system.
